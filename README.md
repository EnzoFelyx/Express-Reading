## üìñ Prelude

This application simulates an online bookstore where you can browse the available books, access their information, search by genre and after choosing your main literary tastes, your purchase is ready to be confirmed in the basket tab.

As my first React-Native project to be developed, the main objective of this application was to test my initial capabilities after everything I learned studying abroad.
